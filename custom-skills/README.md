# Custom Skills & Tools

This folder contains all custom tools, utilities, and scripts developed for this project.

## 📂 Structure

```
custom-skills/
├── ai/                 - AI utilities and helpers
├── automation/         - Automation scripts and workflows
├── experiments/        - Experimental code and features
├── scraping/          - Web scraping tools and utilities
├── utils/             - General-purpose utilities
├── web/               - Web-related tools and functions
├── index.js           - Main entry point
└── README.md          - This file
```

## 🚀 Usage

Start with `index.js` for the main entry point of this project.

Each subdirectory contains related functionality:

- **ai/** - Artificial intelligence related utilities
- **automation/** - Scripts for automating tasks
- **experiments/** - Experimental features being tested
- **scraping/** - Web scraping and data extraction
- **utils/** - Helper functions and common utilities
- **web/** - Web-related functionality

## 📝 Notes

These are custom tools developed specifically for this project.

For Anthropic's official skills and tools, see the `anthropic-skills/` folder.
