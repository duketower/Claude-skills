# web/docs/ — Client-Facing Website Project Documents

Reusable templates for onboarding website clients and scoping projects.

## Files

| File | Purpose | When to Use |
|------|---------|-------------|
| `client-brief-template.md` | Structured brief for Claude to read before building | Fill before starting any new website project |
| `client-intake-form.md` | Simple form to send to the client (business owner) | Send to client to collect content, goals, and preferences |

## Workflow

1. **Send** `client-intake-form.md` to the client (via WhatsApp or email)
2. **Client fills** it with their business details, goals, content
3. **You complete** `client-brief-template.md` using the client's answers + your own research
4. **Paste the brief** into Claude's context at the start of a new session
5. Claude builds the site with full context — minimal back-and-forth

## Works For

All verticals: Preschool, School, Clinic, Salon, Restaurant, Agency, Local Services, etc.
