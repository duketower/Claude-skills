# Web Verticals — Industry-Specific Website Knowledge

Research-backed knowledge base for building websites across different industries.
Each file documents page structure, design conventions, UX patterns, and reference repos
for a specific vertical — so any new site starts with the right foundation, not guesswork.

Built by researching 15+ GitHub repos across all verticals (codewithsadee, Themewagon,
BootstrapMade, HTML5UP, Colorlib) plus extended research on design patterns and standards.

## How to Use in a Claude Session

When starting a new website project, tell Claude:

> "I have a verticals knowledge base at `duketower/Claude-skills` in `web/verticals/`.
> Read the relevant vertical file before building the site for [industry]."

Claude will read the file and immediately know:
- What pages/sections to build
- Which design palette and fonts to use
- What forms and UX patterns to include
- What schema markup to add
- India-specific requirements (WhatsApp, phone formats, etc.)
