# experiments/

A sandbox for ideas, prototypes, and work-in-progress skills.

## What Goes Here

- New API explorations ("can I scrape X with Playwright?")
- Proof-of-concepts before committing to an approach
- Failed attempts — documented so you don't repeat them
- Temporary scripts that might become real skills
- Anything you're not sure where to put yet

## Rules

- **No stability guarantees** — code here may break or be deleted
- **No exports** — nothing in this folder is imported by other skills
- **Add a header comment** explaining what you were trying to do
- **Graduate working experiments** — move to the right folder when ready

## Graduation Checklist

When an experiment is ready to become a real skill:

- [ ] Move file to the correct subfolder (`automation/`, `scraping/`, etc.)
- [ ] Clean up code and add a full header comment
- [ ] Export from that folder's `index.js`
- [ ] Register in root `index.js`
- [ ] Delete or archive this experiment file
